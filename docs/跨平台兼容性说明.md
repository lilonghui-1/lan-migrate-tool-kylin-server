## 核心技术

### 1. 协议层完全跨平台

协议基于 **TCP Socket + 自定义二进制协议**：
- 不依赖任何平台特定的功能
- 消息格式：`[4B 命令类型][4B 载荷长度][N B 载荷数据(JSON UTF-8)]`
- 使用 `struct` 打包/解包，跨平台一致
- 完整支持中文文件名（UTF-8 编码）

### 2. 服务端跨平台

`server/kylin_server.py` 兼容：
- **Windows**（作为客户端接收端）
- **麒麟桌面版**（Kylin OS Desktop）
- **麒麟服务器版**（Kylin OS Server）
- **Ubuntu / Debian / CentOS** 等所有 Linux 发行版
- **macOS**（理论上支持）

### 3. 客户端多端适配

| 平台 | 启动脚本 | 模式 |
|------|----------|------|
| **Windows** | `main.py` | GUI (PyQt6) |
| **麒麟桌面版** | `kylin_gui_client.py` | GUI (PyQt6) |
| **麒麟服务器版** | `server/kylin_cli_client.py` | CLI (命令行) |
| **Linux 服务器** | `server/kylin_cli_client.py` | CLI (命令行) |

## 快速使用

### 场景 A：Windows ↔ 麒麟（任意版本）

**麒麟端（接收/发送方）：**
```bash
python3 server/kylin_server.py -p 9000
```

**Windows 端：**
```bash
py main.py
# 在 GUI 中连接麒麟IP，使用"麒麟传输"页面
```

### 场景 B：麒麟服务器版 ↔ 麒麟服务器版

**服务器B（接收方）：**
```bash
python3 server/kylin_server.py -p 9000
```

**服务器A（发送方）：**
```bash
# 交互模式
python3 server/kylin_cli_client.py <B_IP> -i

# 或单命令模式
python3 server/kylin_cli_client.py <B_IP> --upload /data/file.tar.gz --remote-dir /backup
python3 server/kylin_cli_client.py <B_IP> --download /data/file.tar.gz --local-dir ~/Downloads
```

### 场景 C：麒麟桌面版 ↔ 麒麟服务器版

**服务器版（接收方）：**
```bash
python3 server/kylin_server.py -p 9000
```

**桌面版（发送方）：**
```bash
python3 kylin_gui_client.py
# 在 GUI 中连接服务器IP，使用"麒麟传输"页面
```

## 端口和网络要求

- **协议端口**：TCP 9000（可在 `config.py` 中修改）
- **mDNS 发现端口**：UDP 5353（可选）
- **网络要求**：双方在同一局域网
- **防火墙**：需开放 TCP 9000 端口

## 已知限制

1. **路径分隔符**
   - Windows 使用 `\` 作为路径分隔符
   - Linux/麒麟 使用 `/` 作为路径分隔符
   - 代码已使用 `os.path.join` 自动处理

2. **换行符**
   - 文本文件传输时保留原始换行符
   - 二进制模式传输，不会转换

3. **文件权限**
   - Linux/麒麟端保留原始权限位
   - Windows 端不保留 Linux 权限信息

4. **符号链接**
   - 当前版本不跟随符号链接
   - 符号链接会作为普通文件传输
